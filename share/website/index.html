Algo Language compiler, C code generation

A2C is designed for everyone who wants to compile the abstract Algorithmic Language (especially EPITA students)

A2C is an open source software, based on the C language. Ghom is a group of 4 EPITA Students, which is a French IT Engeenering School. This project is a part of the free schoolar project of the second year, and it's integrally written in C. If you want to know more about us, go to group presentation.
